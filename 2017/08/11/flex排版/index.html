    flex排版

    起手

    外層加上

    display: flex

    初始預設設定

    flex-direction: row 方向

    justify-content: flex-start 主軸分布

    align-items: flex-start 副軸分布

    方向

    flex-direction: row 橫向正向排列

    flex-direction: column 垂直正向排列

    flex-direction: row-reverse 橫向反向排列

    flex-direction: column-revers 垂直反向排列

    主軸分布

    主軸就是跟flex方向相同的方向，假設flex-direction: row (橫向) 此時主軸的方向就是水平的

    justify-content: flex-start 對齊主軸最上方或最左方排起

    justify-content: center 對齊主軸中間

    justify-content: flex-end 對齊主軸最下方或最右方排起

    justify-content: space-around 左右各留一份空間，元素之間是兩份空間

    justify-content: space-between 左右不留空間，元素之間平均分配空間

    justify-content: space-evenly 左右跟元素之間平均分配空間

    副軸分布

    副軸就是跟flex方向垂直的方向，假設flex-direction: row (橫向) 此時副軸的方向就是垂直的

    align-items: flex-start 對齊副軸最上方或最左方排起

    align-items: center 對齊副軸中間

    align-items: flex-end 對齊副軸最下方或最右方排起

    align-items: space-around 左右各留一份空間，元素之間是兩份空間

    align-items: space-between 左右不留空間，元素之間平均分配空間

    align-items: space-evenly 左右跟元素之間平均分配空間

    使用flex控制份數

    可直接使用 flex: $number 控制份數

    html部分

    <div class="flex_wrapper">
   <div class="flex_wrapper--1">
       <p>第一個區塊</p>
    </div>
    <div class="flex_wrapper--2">
        <p>第二個區塊</p>
    </div>
    <div class="flex_wrapper--3">
        <p>第三個區塊</p>
    </div>
 </div>

    css部分

    .flex_wrapper{
    display:flex;
    width: 500px;
}

.flex_wrapper--1{
    flex: 6;
}

.flex_wrapper--2{
    flex: 4;
}

.flex_wrapper--3{
    flex: 4;
}

    在這樣的分配代表第一個 div 的寬度佔 500px / (6 + 4 + 4) * 6 = 214.28px
    亦可設定其一有固定寬度，將Css中的 flex_wrapper–2 修改為以下

    .flex_wrapper--2{
    flex: 250px;
    flex-shrink: 0;
    flex-grow: 0;
}

    在這樣子的分配後”第二區塊”就永遠都會是 250px ， 第一區塊的算法就會變成
    ( 500px - 250px ) / ( 6 + 4 ) * 6 = 150px

    flex-grow 控制膨脹分配

    如果外層寬度超過內層設定長度總和，內層預設是會等比例膨脹，可以透過此css來控制其一內層不要進行膨脹( 重複使用上面的html，css更改為以下 )

    .flex_wrapper{
    display:flex;
    width: 100%;
}

.flex_wrapper--1{
    flex: 500px;
}

.flex_wrapper--2{
    flex: 400px;
}

.flex_wrapper--3{
    flex: 300px;
    flex-grow: 0;
}

    透過上面的css語法，也就是說當外層寬度超過 1200px ( 500px + 400px + 300px ) 時，僅有第一區塊跟第二區塊會等比例放大，而第三區塊會維持在 300 px

    flex-shrink 控制縮減分配

    如果外層寬度小於內層設定長度總和，內層預設是會等比例縮減，可以透過此css來控制其一內層不要進行縮減( 重複使用上面的html，css更改為以下 )

    .flex_wrapper{
    display:flex;
    width: 100%;
}

.flex_wrapper--1{
    flex: 500px;
    flex-shrink: 0;
}

.flex_wrapper--2{
    flex: 400px;
}

.flex_wrapper--3{
    flex: 300px;
}

    透過上面的css語法，也就是說當外層寬度低於 1200px ( 500px + 400px + 300px ) 時，僅有第二區塊跟第三區塊會等比例縮減，而第一區塊會維持在 500 px

    方向排列順序

    一般方向排列都是依照html的順序來進行正向或是反向的排列，可以透過 order 的方式來把其中的一個移動到最前面或是最後面( 重複使用上面的html，css更改為以下 )

    .flex_wrapper{
    display:flex;
    width: 100%;
}

.flex_wrapper--1{
    flex: 500px;
    order: 1000;
}

.flex_wrapper--2{
    flex: 400px;
    order: -1;
}

.flex_wrapper--3{
    flex: 300px;
}

    此時第二區塊就會被移動到最前方，第一區快就會被移到最後方，當進入到RWD設計時可以透過此方式來移動元素的位置，達到更好的排版
