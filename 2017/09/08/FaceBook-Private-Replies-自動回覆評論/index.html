    FaceBook Private Replies 自動回覆評論

    官方說明文件 https://developers.facebook.com/docs/graph-api/reference/v2.10/object/private_replies

    準備事項

    • 粉絲團權限 ( 後面會說明怎麼生產Token )
    • 一台可以回應的Server

    建立應用程式

    網址 https://developers.facebook.com/

    設定應用程式名稱

    設定Messager

    建立Webhooks回呼程式

    設定Webhooks

    1. 輸入回呼網址(https://xxxx.com.tw/aaa.php)(https是必要的)
    2. 輸入verify_token
    3. 勾選訂閱欄位
    4. 驗證並儲存

    產生粉絲專業權杖並綁定Webhooks

    1. 在權杖產生的欄位內點選擇你的粉絲專業
    2. 在Webhooks的欄位選擇你的紛絲專業並按下訂閱

    確認Webhooks訂閱粉絲專頁

    訂閱完成就會出現下面這樣的狀態

    在Webhooks訂閱feed

    選擇左方的 Webhooks 找到 feed -> Subscribe

    發佈你的應用程式

    選擇左方應用程式審查，發佈應用程式

    要求Token權限

    網址 https://developers.facebook.com/tools/explorer/

    1. 應用程式欄位選擇剛剛新增的應用程式
    2. 選擇權杖的地方點選”取得用戶存取權杖”

    1. 選擇Publish_pages
    2. 按下取得存取權杖

    1. 將取得權杖的部分換成你的粉絲專頁

    取得永久Token權限

    你會發現目前的Token是有到期時間的，接下來要將我們的權杖期限延長，避免權杖過期後要一直更換權杖

    1. 點選存取權杖旁邊的藍色驚嘆號
    2. 點擊以”以存取權杖工具開啟”

    1. 點擊”延伸存取權杖”，並將權杖複製起來

    1. 將權杖在上方欄位後並按下偵錯來確認這個權杖的到期日”永不”，並且範圍有剛剛要求的權限”publish_pages”

    1. 回到Graph API( https://developers.facebook.com/tools/explorer/ )
    2. 把剛剛拿到的權杖貼上存取權杖，接下來我們要進行回覆的測試

    發文取得文章的ID

    粉絲團如果有自訂ID跟沒自訂基本上就是以上兩種文章網址，文章ID就如紅色框起來處

    後記

    1. 同一個留言Feed會傳送不只一次，可能會導致重複回覆訊息，可以透過資料庫來做驗證是否已經回覆過了。

    2. 如果要回應圖片可以修改傳送出去的資料多補上attachment_url

    3. 可以透過字串判斷來決定使用這留不同的訊息，自動回覆不同的內容
