    CSS動畫

    大致上說明可以分為transitions和animation；兩者的不同之處在於tansitions是當前後參數不同時觸發改變，animation是ID或是Class將動畫指派給animation就會直接觸發

    Tansition

    狀態改變時間

    動畫從開始到結束進行的時間

    動畫狀態延遲

    後延遲幾秒後才開始動畫，也可以是負值，讓畫面載入之前動畫就先開始動作

    針對特定Class進行設定

    只針對指定Class進行動作，不是針對所有元素進行動作

    動畫動作定位

    從左上角(0,0)設定為動畫原點，動畫進行動作的時候依照此原點進行動作

    Animation

    狀態改變時間

    動畫從開始到結束進行的時間

    動畫狀態延遲

    後延遲幾秒後才開始動畫，也可以是負值，讓畫面載入之前動畫就先開始動作

    動畫影格

    動畫除了靠前後屬性不同進行動作，也可以將動畫指派執行一組影格

    指派給動畫影格

    指派給keyframesname這一個動畫影格

    動畫線性函數

    針對動畫移動的線性設定

    動畫播放次數

    指定動畫播放次數，或是設定infinite無限循環播放

    動畫播放前後狀態

    動畫填充模式，可以設定動畫在開始之前或是在結束之後的動作

    如果碰到Firefox，SVD的定位設定要多上
